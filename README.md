# Telegram Assistant Bot

A personal Telegram assistant bot with user authorization and basic AI responses.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set environment variable `TELEGRAM_BOT_TOKEN` with your bot token.
3. Run the bot: `python bot.py`

## Features
- User Authorization (Only specific User IDs can use the bot)
- Basic Intelligent Responses
- Production-ready for cloud deployment
